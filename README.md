# RandomStudy - iOS 
## 기능정의
* 목표를 달성하기 위해 준비하는 일들을 저장한 후, 매일 불러와 달성하는 앱 입니다.
* 내가 목표를위해 이뤄냈던 일들을 한눈에 찾아 볼 수 있습니다.
